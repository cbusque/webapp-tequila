# webapp-tequila
Get : 
http://localhost:3001/api/actions/action?action=lions 
permet d'obtenir le nombre de vote pour l'aciton lions
peut etre : 
lions;
water;
cheese;
trap;

Post : 
http://localhost:3001/api/inGameEvents/state?state=act1&timer=45
permet de set letat du jeu sur le serveur 
entre : 
   "not_started",
    "act1",
    "act2",
    "act3"
